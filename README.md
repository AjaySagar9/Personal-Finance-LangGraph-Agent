# 💰 Personal Finance LangGraph Agent

An AI-powered Multi-Agent Personal Finance Management System built using **LangGraph**, **Python**, and **LangChain**.

This project demonstrates how multiple AI agents collaborate to analyze income, expenses, savings, and financial health while generating personalized recommendations.

---

# 🚀 Project Overview

Managing personal finances manually can be challenging.

This LangGraph-based Finance Agent automates the process by:

- Tracking expenses
- Calculating total spending
- Calculating savings
- Analyzing financial health
- Generating recommendations
- Producing a final financial report

The system uses a Multi-Agent Architecture where each agent is responsible for a specific financial task.
